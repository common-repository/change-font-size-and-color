Change font style by changing its size and colors for entire website.

== Description ==

It will change h1 size and color, h2 size and color, h3, h4 size and color, p size and color, link size and color, etc. Its very easy to change font style and color of entire website. This will change look and feel of Website fonts by adding size and color to it.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `change-font-size-color.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto settings and you will find plugin there.

== Frequently Asked Questions ==

= How can reset to old style? =

1. By adding exact font and color to this plugin.
2. Deactivate the plugin will change to previous style of website.


= Minimum Requirements =

* WordPress 4.2 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater


= Manual installation =

**Uploading in WordPress Dashboard**

1. Download `Change font size and color`
2. Navigate to the 'Add New' in the plugins dashboard
3. Navigate to the 'Upload' area
4. Select `change-font-size-and-color.zip` from your computer
5. Click 'Install Now'
6. Activate the plugin in the Plugin dashboard
7. Find it under Settings menu

**Using FTP**

1. Download `change-font-size-and-color.zip`
2. Extract the `change-font-size-and-color.zip` directory to your computer
3. Upload the `change-font-size-and-color.zip` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard

The WordPress codex contains [instructions on how to install a WordPress plugin](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
